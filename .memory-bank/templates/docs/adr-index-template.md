## How to Use This Index

1. Each ADR must be listed here chronologically
2. Link to the actual ADR file
3. Keep status updated (Proposed → Accepted → Deprecated/Superseded)
4. Cross-reference with related tasks and other ADRs

## ADR Naming Convention

- Format: `ADR-###-kebab-case-title.md`
- Sequential numbering starting from 001
- Use descriptive titles
